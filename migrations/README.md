# Миграции БД

Скрипты применяются вручную по порядку (по номеру в имени).  
Перед применением сделайте бэкап БД.

## Как применять

- **SQLite:** `sqlite3 schedule.db < migrations/001_avatar_column.sql`
- **PostgreSQL:** `psql $DATABASE_URL -f migrations/001_avatar_column.sql`

Или выполнить SQL через любой клиент (DBeaver, pgAdmin и т.д.).

## Уже встроено в приложение

Таблицы создаются в `app.database.init_db()`. Эти файлы — для документации и ручных правок (например, если БД была создана старой версией и нужна только одна миграция).
